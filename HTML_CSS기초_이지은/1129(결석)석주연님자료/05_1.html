div 요소를 이용하여 레이아웃하게 되면 div 태그가 많이 사용된다.
페이지 내에 산재되어 있는 div 태그들은 html 문서를 복잡하게 만들고 가독성이 떨어진다.
이러한 단점을 보완하기 위해 태그 자체가 의미를 가지는 시멘틱(Semantic) 태그가 추가되었다.
<header><nav><section><article><aside><footer> 등이 있다.

header 웹 페이지 상단에 위치한 로고, 상단 메뉴, 메인 메뉴 등이 들어가 있는 영역
nav 네비게이션 링크가 들어가는 영역 (보통 header 내에 있음)
section 메인 컨텐츠와 같이 독립적이 섹션을 나타낼 때 사용
article section 내에 들어가는 내용들을 세부적을 나누어서 정리
aside section 좌측 또는 우측의 사이드바에 사용. 부가적인 요소의 영역
footer 회사나 기관의 주소, 연락처, 저작권 표시 등의 요소를 포함
